# 🌦️ ClimateIQ – AI Weather Intelligence Dashboard

ClimateIQ is an advanced AI-powered weather intelligence platform that delivers real-time weather analytics, air quality insights, climate scoring, weather forecasting, multi-city comparisons, and personalized weather recommendations through an interactive dashboard.

---

## 🚀 Project Overview

ClimateIQ transforms traditional weather forecasting into an intelligent decision-support system. By combining real-time weather data, air quality monitoring, predictive analytics, and AI-generated insights, the platform helps users make informed decisions regarding travel, outdoor activities, health, and daily planning.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ClimateIQ.git
```

### Navigate to Project

```bash
cd ClimateIQ
```

### Configure API Key

Open `script.js` and replace:

```javascript
const API_KEY = "YOUR_OPENWEATHER_API_KEY";
```

with your OpenWeatherMap API key.

### Run Application

Simply open:

```text
index.html
```

in your browser.
